# PECS
PECS (Pre-Epistemic Cognitive System) - implies "muscle" or "building strength." This fits the learning approach (neurons that fire together, wire together), where connections are strengthened over time.

The purpose of this system is to dramatically reduce token usage when working with LLms by Doing the thinking and Linking part of the Cognitive process an donly Relying on LLM as a translation and communciation layer.
